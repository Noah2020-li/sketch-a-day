Hexagons! 

```python
def hexagon(x, y, r):
    with pushMatrix():
        translate(x, y)
        beginShape()
        for i in range(6):
            vx = cos(i * TWO_PI /6) * r
            vy = sin(i * TWO_PI /6) * r
            vertex(vx, vy)
        endShape(CLOSE)
```
